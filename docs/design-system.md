# Design System

Colors:

- Purple: `#5B2C91`
- Purple Dark: `#321046`
- Orange: `#F5A623`
- Orange Deep: `#E77F00`
- Soft Blue: `#B9D2EA`
- White: `#FFFFFF`
- Light Gray: `#F5F5F7`

Principles:

- Arabic-first RTL.
- Purple for command, navigation and premium cards.
- Orange for primary CTAs and offers.
- Soft gray/white for operations and admin clarity.
- Every operational screen shows status, next action, owner, timestamp and audit visibility.
